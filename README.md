 #SocialApp

هذا تطبيق تواصل اجتماعي بسيط قمت ببنائه باستخدام Qt6 و QML لنظام أندرويد.

 ##محتويات المشروع:
- C++: يحتوي على معالج الكاميرا وإدارة قاعدة البيانات (SQLite).
- QML: يحتوي على واجهة المستخدم الرئيسية وصفحات التطبيق.

## حالة البناء:
يتم بناء التطبيق تلقائياً باستخدام GitHub Actions عند كل تحديث للكود.  _
